# Twitter Sentiment Analysis (Python)

This project builds a multi-class sentiment classification model for Twitter data. The workflow includes text preprocessing, feature extraction, model comparison, and detailed performance evaluation across multiple machine learning algorithms.

## Tech Stack
- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib & Seaborn  
- NLP preprocessing & vectorization  

## Models Implemented
- Logistic Regression  
- Naive Bayes  
- Random Forest  
- K-Nearest Neighbors (KNN)  

## Key Work
- Text cleaning and preprocessing  
- TF-IDF / vectorization for feature extraction  
- Multi-class classification (Irrelevant, Negative, Neutral, Positive)  
- Model comparison using precision, recall, F1-score  
- Per-class performance visualization  
- Best model achieving ~91% accuracy  

The notebook demonstrates a complete end-to-end NLP pipeline with comparative model evaluation.
